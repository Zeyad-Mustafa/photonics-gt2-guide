# Issue Template — New Section or File Request

Use this template when you want to request new content that does not currently exist in the guide — a missing topic, a new workflow, or a new application file.

---

## What Is Missing

Describe the topic, procedure, or information that is not covered in the current guide.

---

## Which Section Would It Belong In

Which existing section folder is the most appropriate home for this content?
Example: 08_Development_and_Post_Processing/ or 10_Applications/

If it does not fit any existing section, explain why a new section might be needed.

---

## Why This Is Useful

Who would benefit from this content and in what situation? One to three sentences is sufficient.

---

## What You Already Know About This Topic

If you have direct experience with the topic, briefly describe it here. This helps the maintainer evaluate whether you are the right person to write the content, or whether it should be researched and written centrally.

---

## Are You Willing to Write This Content

Mark one:

- Yes — I can write a draft and submit a pull request
- Partially — I can provide the raw information but would need help with formatting
- No — I am flagging the gap only; I am not able to write it myself
