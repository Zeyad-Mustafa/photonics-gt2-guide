# Issue Template — Correction

Use this template when you have found a factual error, an outdated value, or a misleading statement in the guide.

---

## File and Section

Which file contains the error?
Example: 08_Development_and_Post_Processing/development_times_table.md

Which section or heading within that file?
Example: IP-S development times table, row for structures 100 to 300 um tall

---

## What the Current Text Says

Paste the sentence or value that is incorrect.

---

## What It Should Say

Provide the corrected text or value.

---

## Why This Is Wrong

Brief explanation of why the current text is incorrect. If you are correcting a parameter value, describe where the correct value comes from (your own experience, a Nanoscribe application note, a published paper, etc.). You do not need to provide a formal citation.

---

## How Serious Is This Error

Mark one:

- Critical — following the current text could damage equipment, harm a sample, or create a safety risk
- Significant — following the current text will likely produce a failed print or incorrect result
- Minor — the error is factual but unlikely to cause a practical problem (wrong unit, typo, outdated software version name, etc.)
