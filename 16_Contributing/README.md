# Section 16 — Contributing

## Overview

This guide was built to be a living document. The GT2 is a complex machine, resin formulations change between lots, facilities have different configurations, and new applications appear in the literature regularly. No single person can keep all of that current alone.

If you have used the GT2, found an error in this guide, have a result worth sharing, or know something that is missing — contributions are welcome.

## What Kind of Contributions Are Welcome

Corrections to factual errors — wrong development times, incorrect parameter values, outdated resin compatibility information. These are the highest priority.

Additional print results — SEM images, optical microscope images, or photographs of printed structures that illustrate a concept in the guide. Images must be your own work or properly licensed.

New application examples — optional additions to Section 10. If you have printed something notable that is not covered, you are welcome to propose a new application file.

Workflow improvements — if you have found a better procedure for substrate preparation, interface detection, or development for a specific resin, and you can document it clearly, it is welcome.

Clarifications — if a section is confusing, ambiguous, or assumes knowledge it should not assume, opening an issue describing the problem is a useful contribution even without a proposed fix.
