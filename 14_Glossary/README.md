# Module 14 — Glossary

> This module defines every term used in this guide that may be unfamiliar to a new GT2 operator. Terms are organized alphabetically in a single file for easy searching.

---

## How to Use

All definitions are in [full_glossary.md](./full_glossary.md).

If you encounter an unfamiliar term anywhere in the guide, search this file first. Each definition includes a cross-reference to the module where the concept is covered in depth.

The glossary covers:

- Optical and microscopy terms (NA, WD, voxel, focal spot, aberration)
- Two-photon polymerization physics (2PP, TPA, photoinitiator, threshold, dose)
- Nanoscribe-specific terms (DiLL, SPS, GWL, DeScribe, NanoWrite, IP-series)
- Fabrication process terms (STL, slicing, hatching, development, shrinkage, CPD)
- Resist and materials terms (acrylate, monomer, crosslinking, n, delta-n)
- Scanning and stage terms (piezo, galvo, stitching, tilt correction)
- Post-processing terms (debinding, sintering, brown body, UV curing)
- Safety terms (Class IV, OD, SPS, E-stop, PGMEA, sensitizer)
