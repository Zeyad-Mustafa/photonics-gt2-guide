# 🔧 25× Objective — Installation Steps

---

## Before You Begin — Safety Checklist

- [ ] Green "Process" light is **OFF**
- [ ] Clicked **"Exchange Holder"** in NanoWrite
- [ ] Clicked **"Open"** and waited for button to flash
- [ ] Adjustment ring set to **"Glyc"** position (3 longest bars)

---

## Step-by-Step Installation

### Step 1 — Open the Enclosure
In NanoWrite: **"Exchange Holder"** → **"Open"** → wait for green flash → lift lid.

### Step 2 — Remove Any Existing Objective
Rotate counterclockwise to remove → place in labeled protective case immediately.

### Step 3 — Set the Adjustment Ring
Before installing, rotate the collar on the 25× body to the **"Glyc"** position:
- Look for the scale on the collar
- Align the index mark with the **3 longest bars** (Glyc position)
- Confirm with a gentle click/detent

### Step 4 — Attach the Silicone Ring
The 25× uses a **silicone ring** (same as 10×, different from the 63× fiber ring):
1. Slide the silicone ring onto the 25× objective body
2. Position it at the correct stop location (it should sit snugly at the marked position)

### Step 5 — Install the 25× Objective
1. Hold **by the body** — never touch the front glass element
2. Insert into the turret slot
3. Rotate **clockwise until it stops** — do not overtighten

### Step 6 — Configure in NanoWrite
1. Touchscreen controller: select objective position for 25×
2. NanoWrite: set objective to **"25×"**
3. Set holder type to **"ITO-coated glass"** (or silicon if using silicon substrate)

### Step 7 — Close and Resume
Lower the lid → click **"Close"** in NanoWrite → green light comes back on.

---

## Pre-Print Verification

Before running your job, confirm:
- [ ] Adjustment ring is at "Glyc" — check visually
- [ ] Resin drop is semi-spherical (not flat)
- [ ] ITO side of substrate is facing DOWN (toward objective)
- [ ] Correct resist selected in DeScribe job file

---

## Common Mistakes

| Mistake | Consequence | Prevention |
|---|---|---|
| Adjustment ring at wrong position | Blurry voxels, failed prints | Check ring EVERY session |
| ITO side facing up instead of down | Poor interface detection, aberrated printing | Mark substrate with pen on non-ITO edge |
| Using fiber ring instead of silicone ring | Wrong standoff height | Check ring type label before installing |
| Forgetting to update objective in NanoWrite | Software uses wrong parameters | Always confirm in software after hardware change |
