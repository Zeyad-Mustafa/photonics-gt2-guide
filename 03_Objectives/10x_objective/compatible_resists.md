# 🧪 10× Objective — Compatible Resists

---

## Compatible Resists

| Resist | Status | Notes |
|---|---|---|
| **IP-Q** | ✅ Primary | Best choice for large fast prints |
| **IPX-Q** | ✅ Compatible | Slightly better resolution than IP-Q |
| **IPX-Clear** | ✅ Compatible | For optically transparent large structures |
| IP-Dip2 | ❌ Not compatible | For 63× only |
| IP-S | ❌ Not compatible | For 25× only |
| IP-Visio | ❌ Not compatible | For 25× only |
| IP-PDMS | ❌ Not compatible | For 25× only |

---

## IP-Q — The Default Choice for 10×

**IP-Q** is the standard resist for the 10× objective and is formulated specifically for large-volume, fast printing.

Key properties:
- Refractive index: n = 1.48 @ 780 nm
- Low viscosity → easy to apply and spread
- Optimized sensitivity for low-NA objectives
- Slightly lower resolution than IPX-Q
- Available in larger volumes (cost-effective for big prints)

**How to apply IP-Q with the 10×:**
1. Place substrate (silicon wafer piece) on the sample holder
2. Using the provided pipette, deposit **only 1–2 drops** onto the polished side
3. The drop should sit as a **semi-sphere** — DO NOT let it spread flat
4. A flat drop makes interface detection unreliable
5. Proceed immediately — do not let it sit more than 15–20 minutes

⚠️ **Critical:** With the 10× and IP-Q, only use the **Multi-DiLL Si Wafer holder** configuration in NanoWrite. Using the wrong holder setting will cause incorrect interface detection.

---

## IPX-Q — When You Need Slightly Better Resolution

**IPX-Q** is an improved formulation of IP-Q:
- Slightly higher resolution than standard IP-Q
- Mild coloration (slight tint) in thick structures
- Same application method as IP-Q
- Same substrate compatibility (silicon wafers)

Use IPX-Q when your 10× print requires the best possible feature definition while still using the 10× objective.

---

## IPX-Clear — For Optically Transparent Large Structures

**IPX-Clear** is compatible with all three DiLL objectives including the 10×.

- Highly transparent after curing (> 90% transmission in visible spectrum)
- Best choice when optical transparency matters (waveguides, optical diffusers, light guides)
- Can also be used with 25× and 63×
- Same application method as IP-Q
