# 🔬 63× Objective — Overview

> The 63× is the GT2's highest-precision instrument. When you need to print features at the nanoscale — sub-200 nm walls, photonic crystals, or ultra-smooth optical surfaces — this is the only choice.

---

## Quick Specs

| Property | Value |
|---|---|
| Magnification | 63× |
| Numerical Aperture (NA) | 1.4 |
| Working Distance (WD) | 360 µm |
| Immersion type | DiLL (resin) or Oil Immersion (through glass) |
| Resin stop type | **Fiber ring** (different from 10× and 25×) |
| Lateral voxel size | ~160 nm |
| Axial voxel size | ~500 nm |
| Max structure height | ~360 µm |
| Max structure volume | ~0.1 mm³ |
| Typical laser power | 15–35% |

---

## What Makes the 63× Special

The 63× has **NA = 1.4** — the highest numerical aperture of all GT2 objectives. High NA means:

- Smallest possible focal volume → smallest possible voxels → highest resolution
- Oil immersion (n = 1.515) allows NA > 1.0 — impossible in air
- Tightest focus → most efficient two-photon absorption → lowest required laser power
- The quadratic I² threshold effect is sharpest → features can go below the diffraction limit

The 63× can print features as small as **~160 nm laterally** — smaller than the wavelength of visible light.

---

## The Fiber Ring — Critical Difference

⚠️ The 63× uses a **fiber ring**, NOT the silicone ring used by the 10× and 25×.

The fiber ring is a thin optical fiber looped around the objective tip. It serves as both the standoff spacer and as a light guide for the interface detection system.

**Never install a silicone ring on the 63×.** It will set the standoff height incorrectly and prevent the interface finder from working.

---

## Typical Applications

- Sub-wavelength photonic crystals and metamaterials
- High-resolution micro-optical components (lenses, beam shapers)
- Nanoscale mechanical structures (springs, resonators)
- Waveguides and photonic interconnects
- Structures requiring surface roughness < 10 nm Ra
- Research requiring the absolute minimum feature size

---

## What You Cannot Do With the 63×

- ❌ Structures larger than ~0.1 mm³ in reasonable time
- ❌ Print heights taller than ~360 µm
- ❌ Use IP-S, IP-Visio, IP-PDMS, or IP-Q resists
- ❌ Print on ITO-coated glass in standard DiLL mode (use quartz or silicon)
