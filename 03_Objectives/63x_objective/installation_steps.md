# 🔧 63× Objective — Installation Steps

---

## Before You Begin — Safety Checklist

- [ ] Green "Process" light is **OFF**
- [ ] Clicked **"Exchange Holder"** in NanoWrite
- [ ] Clicked **"Open"** and waited for button to flash
- [ ] **Fiber ring** is ready (NOT silicone ring)
- [ ] IP-Dip2 resin is at room temperature

---

## Step-by-Step Installation

### Step 1 — Open the Enclosure
NanoWrite: **"Exchange Holder"** → **"Open"** → wait for green flash → lift lid.

### Step 2 — Remove Any Existing Objective
Rotate counterclockwise to remove → store in labeled case immediately.

### Step 3 — Attach the Fiber Ring to the 63×
1. Take the fiber ring from its protective case
2. Slide it onto the front of the 63× objective
3. Seat it at the correct stop position
4. Verify the fiber loop is symmetric and undamaged
5. Connect the fiber cable end if required by your lab setup

### Step 4 — Install the 63× Objective
1. Hold **by the body** — never touch the front glass element or the fiber
2. Insert into the turret slot
3. Rotate **clockwise until it stops** — very gentle — do not overtighten
4. Confirm it is firmly seated

### Step 5 — Configure in NanoWrite
1. Touchscreen: select position for 63×
2. NanoWrite: objective → **"63×"**
3. Holder type: **"DiLL"** with your substrate type (quartz or silicon)
4. Mode: **"3D SF DiLL"** for standard printing

### Step 6 — Close and Resume
Lower lid → click **"Close"** → green light comes back on.

---

## Post-Session Cleanup (Critical for 63×)

After every session with the 63×:
1. Remove sample and clean residual resin from fiber ring with lens tissue + IPA
2. Check fiber ring for damage
3. If oil immersion was used: clean front element with lens tissue — never leave oil to dry on the objective
4. Store objective in its case with fiber ring attached in storage position

---

## Common Mistakes

| Mistake | Consequence | Prevention |
|---|---|---|
| Using silicone ring instead of fiber ring | Wrong height, interface detection fails | Check ring type before every install |
| Too high laser power to start | Severe overexposure, blooming, merged features | Start at 15% and calibrate up |
| Resin drop too flat | Interface not detected | Apply fresh semi-spherical drop |
| Dirty front element from previous session | Aberrated focus, poor resolution | Clean after every session |
| Not warming resin before use | Viscosity too high → flat drop → detection fails | 15 min room temp before use |
