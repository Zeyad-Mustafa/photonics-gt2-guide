# Section 15 — Resources and Further Reading

## Overview

This section collects the most useful external resources for GT2 users at every level — from foundational physics papers to application-specific literature to practical online communities. It is organized by topic so you can go directly to the resources most relevant to your current need.

All resources listed here are either freely accessible, widely available through institutional library access, or are official documentation from Nanoscribe. Web links were verified at the time of writing but may change — search by title or DOI if a link does not resolve.

## How to Use This Section

If you are new to the GT2 and want to understand the physics before your first session, start with foundational_papers.md and textbooks_and_review_articles.md.

If you are starting a new project in a specific application area, go directly to the literature file for that application. Each file summarizes what the cited work demonstrates so you can identify the most relevant references without reading every paper first.

If you need help with the machine or software, start with official_nanoscribe_resources.md.

If you are writing a paper or grant proposal that involves GT2 printing, the foundational papers and review articles provide the standard citations for two-photon polymerization as a fabrication method.
