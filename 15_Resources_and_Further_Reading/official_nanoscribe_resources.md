# Official Nanoscribe Resources

## Nanoscribe Website

Main site: https://www.nanoscribe.com

The Nanoscribe website is the primary source for:
- Current product specifications for the GT2 and newer systems
- Application notes organized by field (optics, microfluidics, biomedical, photonics)
- Press releases on published research using Nanoscribe systems
- News on new resin releases and software updates

The specifications on the website reflect the current product generation. Some specifications may differ from your specific machine if it was installed under an earlier software or hardware revision — always cross-reference with the documentation that shipped with your system.

---

## Nanoscribe User Portal

URL: https://www.nanoscribe.com/en/service/customer-portal (requires account registration)

The user portal provides registered customers with:
- Software downloads (DeScribe, NanoWrite updates)
- Detailed resin technical data sheets with lot-specific properties
- Application-specific printing parameter recommendations
- Webinar recordings from Nanoscribe application scientists
- Firmware update packages

Registration requires your institution's machine serial number. Contact your facility manager if you do not have portal access.

---

## DeScribe and NanoWrite Documentation

DeScribe and NanoWrite each ship with a built-in help system accessible through the Help menu in the software. The in-software documentation covers:

- All menu items and their functions
- Parameter definitions and valid ranges
- GWL command reference (NanoWrite help)
- Troubleshooting tables for common errors

The in-software help is the most reliable reference for software-specific questions because it corresponds to the exact version of the software installed on your machine. Online documentation may describe a different software version.

---

## Nanoscribe Application Notes

Application notes are short technical documents (typically 4 to 8 pages) published by Nanoscribe covering specific use cases. They include recommended parameters, workflow tips, and representative results. Available through the user portal and sometimes freely on the website. Particularly useful notes include:

- Micro-optics with Photonic Professional GT2
- IP-Dip2 resin introduction and parameter guide
- GP-Silica workflow and sintering protocol
- Printing on fiber end faces
- Microfluidic device fabrication
- Biocompatible scaffolds with IP-Visio

Search the portal by resin name or application area to find the most relevant notes for your project.

## Nanoscribe Technical Support

For machine faults, software errors, or questions that cannot be resolved through the documentation, contact Nanoscribe technical support:

Email: support@nanoscribe.com
Portal ticket system: available through the customer portal (preferred for issues requiring file attachments or detailed back-and-forth)

When contacting support, include:
- Machine serial number
- NanoWrite and DeScribe version numbers
- Description of the problem with as much detail as possible
- Screenshots of any error messages
- GWL files if the issue is job-specific

Response times are typically 1 to 3 business days for standard inquiries. For urgent issues affecting a time-sensitive experiment, flag the ticket as urgent.

## Nanoscribe Webinars

Nanoscribe holds periodic webinars covering new applications, new resins, and workflow optimization. Past webinars are recorded and available through the customer portal. Webinars are typically 45 to 60 minutes and include live Q and A.

Topics from past webinars have included:
- Introduction to IP-Dip2 and parameter optimization
- GP-Silica: from printing to glass
- Advanced GWL scripting
- Micro-optics on fiber tips
- Biomedical applications and IP-Visio
- Large-volume printing with IP-Q and 10x

Check the Nanoscribe website or portal for upcoming webinar schedules.

## Nanoscribe on ResearchGate and Google Scholar

Nanoscribe maintains a presence on ResearchGate and Google Scholar that aggregates publications from customers using Nanoscribe systems. Searching "Nanoscribe" or "Photonic Professional GT2" on Google Scholar retrieves several hundred papers covering every major application area. This is a fast way to find published parameter sets and results for a specific application before committing to extensive trial-and-error calibration.
