# Section 13 — Key Parameters Reference

## Overview

This section is a quick-reference resource. Unlike the earlier sections that explain concepts in depth, this section is organized for fast lookup during a print session — when you need to confirm a development time, check a compatible substrate, verify a starting laser power, or remind yourself of a critical warning before proceeding.

Every table in this section consolidates information from the detailed files elsewhere in the guide. If you need to understand why a value is what it is, follow the cross-reference links to the relevant section.

## How to Use This Section

During a print session, keep this section open on a second screen or printed out. The most commonly needed references during a session are:

Before DeScribe: printing_parameters_by_objective.md and resin_compatibility_matrix.md

Before loading the substrate: substrate_quick_reference.md

After printing: development_quick_reference.md

Before any step: critical_warnings_checklist.md

---

All values in this section are starting points and typical ranges. Calibrate for your specific resin lot, machine state, and structure geometry using test prints before committing to critical samples.
