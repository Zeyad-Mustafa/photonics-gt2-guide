# Step 4 — Install the 10x Objective

> The 10x objective is the simplest to install of the three DiLL objectives. It uses neither a fiber ring (63x) nor an adjustment ring (25x). However, it requires the Multi-DiLL Silicon Wafer Holder — and must never be used with the Universal Sample Holder.

---

## The Holder Compatibility Rule

The Multi-DiLL Silicon Wafer Holder is recessed to accommodate the 10x objective's longer working distance (700 um). If you use the Universal Holder with the 10x objective, the objective will reach its working distance before the stage is in position — or worse, it will crash into the substrate.

The rule is absolute: **10x objective = Multi-DiLL Silicon Wafer Holder. No other holder.**

---

## Procedure

1. In NanoWrite, use the Exchange Holder dialog to retract the stage to the safe exchange position.
2. Open the top hatch.
3. Thread the 10x objective into the appropriate turret position, finger-tight. Verify the turret position is correct for the 10x.
4. Close the hatch.
5. In NanoWrite, select the 10x in the objective dropdown.
6. Verify the working distance is set to 700 um.
7. Confirm the Multi-DiLL Silicon Wafer Holder is loaded (or will be loaded at the next step — the holder with IP-Q applied).

---

## After Printing — Cleaning the 10x

IP-Q is less viscous than IP-S but slightly more viscous than IP-Dip2. Clean the 10x objective tip after each print:
1. Lens tissue with PGMEA — one gentle wipe across the front element.
2. Fresh lens tissue with IPA — follow immediately.
3. Inspect and repeat if needed.
