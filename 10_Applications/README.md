# Section 10 — Applications

## Overview

The GT2 is a general-purpose nanofabrication tool. Its ability to produce true 3D structures with sub-micron resolution and smooth surfaces makes it useful across a wide range of research fields. This section documents the most established application areas, explaining what the GT2 contributes to each field, what design and material choices are most appropriate, and what results have been demonstrated in published research.

Each file in this section follows the same structure: a description of the application area, why the GT2 is suited to it, the specific objectives and resists most commonly used, the design considerations unique to that application, representative results from the literature, and limitations of the GT2 for that specific use case.

This section is intended to help researchers who are new to the GT2 understand whether their application is a good fit, and to give experienced users a reference for comparing approaches across disciplines.

## Choosing the Right Application File

If you are new to the GT2 and exploring whether it fits your research, start with the application file closest to your field. Each file explains the minimum feature sizes required, which objective and resist combination is standard for that application, and what the typical end-to-end workflow looks like.

If your application does not appear in this list, it does not mean the GT2 cannot be used. The underlying capability — true 3D printing with sub-micron resolution in a range of polymer and composite materials — applies broadly. Use the application files as a starting point and adapt the closest matching workflow to your specific needs.
