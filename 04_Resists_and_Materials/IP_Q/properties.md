# IP-Q — Properties

---

## Optical Properties

| Property | Value |
|---|---|
| Refractive index (n) at 780 nm | 1.48 |
| Transmission (cured, visible) | >85% |
| Appearance (cured) | Slightly yellow-tinted, transparent |
| Fluorescence background | Moderate-high |

The slight yellow tint of IP-Q (more pronounced than IP-S) makes it unsuitable for optical applications requiring high visible-light transparency. For optical clarity, use IPX-Clear instead.

---

## Physical Properties

| Property | Value |
|---|---|
| Viscosity | Low-medium (~150-300 mPa·s) |
| Density | ~1.1 g/cm3 |
| Appearance (liquid) | Clear to slightly yellow liquid |

---

## Print Process Properties

| Property | Value |
|---|---|
| Polymerization threshold (typical) | ~10-25% laser power (10x objective) |
| Minimum lateral feature size | ~2-5 um |
| Minimum axial feature size | ~10-20 um |
| Voxel aspect ratio | ~1:5 to 1:8 (10x has largest aspect ratio of all objectives) |
| Scan speed | Up to 100 mm/s (galvo) |
| Slice distance (typical) | 1-5 um |
| Hatch distance (typical) | 1-3 um |

The large voxel aspect ratio of the 10x (lateral ~2-5 um, axial ~10-20 um) means that very thin horizontal features (< ~15 um tall) are difficult to print accurately. For thin structures, prefer the 25x or 63x.

---

## Mechanical Properties (Cured)

| Property | Value |
|---|---|
| Young's modulus | ~0.8-1.5 GPa |
| Shrinkage (linear) | ~5-8% |
| Surface roughness | Higher than IP-S (larger voxel = rougher step edges) |

---

## Developer Compatibility

| Solvent | Role |
|---|---|
| PGMEA | Primary developer (20-30 min) |
| IPA | Rinse (5-10 min) |
