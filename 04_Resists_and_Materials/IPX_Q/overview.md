# IPX-Q — Overview

> IPX-Q is an improved formulation of IP-Q with better optical clarity and slightly finer resolution. If you are choosing between IP-Q and IPX-Q for a large-scale print, this file explains when the upgrade matters.

---

## What Is IPX-Q?

**IPX-Q** is part of Nanoscribe's extended (IPX) series, which represents improved reformulations of the original IP resists. IPX-Q is the extended version of IP-Q, designed for use with the **10x objective** (and in some configurations the 25x) for large-volume printing.

Improvements over IP-Q:
- **Better optical clarity** — less yellowing; cured structures are more transparent
- **Slightly improved resolution** — smaller feature sizes achievable compared to IP-Q
- **Broader objective compatibility** — IPX-Q can be used with both the 10x and 25x objectives

---

## Key Specifications

| Parameter | Value |
|---|---|
| Refractive index (n) | ~1.48 |
| Recommended objectives | 10x and 25x |
| Printing mode | DiLL |
| Appearance (cured) | Clear with slight amber tint (less than IP-Q) |
| Developer | PGMEA, followed by IPA |

---

## When to Use IPX-Q Instead of IP-Q

Use IPX-Q when:

- You are printing with the 10x and optical transparency of the final structure matters
- You want slightly better resolution than IP-Q without switching to IP-S (which requires the 25x)
- You are using the 25x but want a faster, large-field print with acceptable resolution
- Your application can tolerate a slight amber tint but needs better clarity than IP-Q

Continue using IP-Q when:

- You have an existing calibrated workflow with IP-Q and do not need the clarity improvement
- Your lab has established IP-Q parameters that you want to maintain for reproducibility
- Cost is a consideration (IP-Q is typically less expensive)
