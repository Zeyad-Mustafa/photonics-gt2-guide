# IPX-Clear — Overview

> IPX-Clear is the most optically transparent resist in the GT2 lineup and the only one compatible with all four objectives. It is the first choice for any application where light must travel through the printed structure.

---

## What Is IPX-Clear?

**IPX-Clear** is an extended-series photopolymer resist engineered for **maximum optical transparency** across the visible and near-infrared spectrum. It was developed specifically for applications in micro-optics, photonics, and optical sensing where the printed material must transmit, guide, or focus light.

Key distinguishing properties:
- **Highest optical clarity** of all GT2 resists — nearly colorless when cured
- **Compatible with 10x, 25x, and 63x objectives** — the only resist with this breadth
- **Low autofluorescence** — suitable for fluorescence-adjacent applications
- **Refractive index of ~1.50** — well-matched to common optical glasses

---

## Key Specifications

| Parameter | Value |
|---|---|
| Refractive index (n) at 780 nm | ~1.50 |
| Compatible objectives | 10x, 25x, 63x |
| Printing mode | DiLL (all objectives), Oil immersion (63x) |
| Optical transmission (visible) | >95% (among the highest of all IP/IPX resists) |
| Autofluorescence | Low |
| Developer | PGMEA, followed by IPA |
| Shrinkage | Low |

## When to Use IPX-Clear

Use IPX-Clear when:

- You are printing lenses, beam splitters, diffusers, or other refractive optical elements
- Your structure must guide light (waveguides, optical fibers, photonic interconnects)
- You need the lowest possible absorption or yellowing in the printed part
- You want to use one resist across multiple print scales (e.g., design a multi-scale optical system with 10x and 63x elements using the same material)
- Near-IR or visible light transmission through the printed structure is part of the device function

Do NOT use IPX-Clear when:

- Biocompatibility is required (use IP-Visio)
- Flexibility is required (use IP-PDMS)
- You need the absolute minimum voxel size and have already calibrated IP-Dip2 (IP-Dip2 has a better established resolution record for sub-200 nm work)

---

## Multi-Objective Compatibility: A Unique Advantage

No other standard GT2 resist is compatible with the 10x, 25x, and 63x objectives simultaneously. IPX-Clear's n = ~1.50 is close enough to the immersion requirements of each objective to enable interface detection and proper optical performance across all three.

This makes IPX-Clear the natural choice for **multi-scale photonic devices** where:
- A large scaffold or support structure is printed with the 10x
- Medium-resolution waveguide routing is printed with the 25x
- High-resolution coupling features (gratings, nano-antennas) are printed with the 63x
- All in the same material, with no refraction mismatch at interfaces
